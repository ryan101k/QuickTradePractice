# QuickTradePractice

주식 모의투자와 선택형 인생 시뮬레이션을 결합한 브라우저 게임입니다.

빠르게 움직이는 가상 시장에서 주식·ETF·경제자산을 거래하고, 매달 장을 마감한 뒤 직업, 인간관계, 연애, 가족, 부동산, 대출 같은 인생 행동을 선택합니다. 실제 투자 서비스가 아닌 학습·게임용 프로젝트이며 모든 기업과 가격은 가상입니다.

## 게임 흐름

1. 나래의 튜토리얼을 진행하거나 건너뜁니다.
2. 첫 직업을 선택합니다. 직업을 고르기 전에는 장을 열 수 없습니다.
3. `장 열기`를 누르면 한 달의 시장이 시작됩니다.
4. 제한된 장중 시간 동안 종목을 분석하고 주문합니다.
5. `장 마감` 후 월급·대출이자·보유자산·관계 변화를 정산합니다.
6. 남은 행동력으로 데이트, 취미, 이직, 인맥, 부동산, 법정 대응 등을 선택합니다.
7. 다음 달 장을 열어 경제와 인생을 이어갑니다.

게임의 1회 장 개장·마감은 인생 시간으로 한 달입니다.

## 주요 기능

### 주식과 경제

- 11개 기업 섹터와 대·중·소형 가상 종목
- 일반·레버리지·인버스 ETF
- 금현물, 국채 10년, 달러선물 거래
- 회복기, 호황, 과열, 긴축, 침체, 금융위기, 부양 국면의 경제 사이클
- 기준금리와 물가상승률 변화
- 금리 변화가 대출이자·채권·금·달러·주식에 미치는 영향
- 기업·섹터·시장 뉴스와 희귀 긴급 뉴스
- 기업 파산·상장폐지·재상장 및 발생 원인 팝업
- 월말 주요 뉴스 선택 열람과 전문가 의견

경제자산의 움직임은 대표적인 관계를 학습하기 위해 단순화했습니다. 실제 시장에서는 정책 기대, 환율, 수급 등 여러 요인이 동시에 작용합니다.

### 주문과 차트

- 수량 또는 금액 기준 매수·매도
- 공매도와 레버리지 구분
- 평균 매입가, 평가손익, 실현손익 표시
- 시가·고가·저가·종가 캔들 및 라인 차트
- 최고가·최저가·내 평단가 기준선
- 포트폴리오 구성과 순자산 추이
- 매매 수수료 0.015%, 매도 거래세 0.18%
- 일일 상·하한가 ±30%

### 인생 시뮬레이션

- 직업, 경력, 승진, 자격증, 이직 및 직업별 사고 위험
- 월급, 세금, 보험, 연금, 신용등급과 대출
- 주거와 투자용 부동산
- 건강, 행복, 스트레스, 체력, 노화와 상속
- 취미와 월별 행동력
- 인맥 형성, 신뢰, 부탁과 역할별 효과
- 범죄·수사·변호사·재판·판결
- 출산·입양·자녀 교육과 가족 유대
- 사망 이후 세대 계승과 인생 연대기

### 연애와 인간관계

- 소개팅 앱, 지인 소개, 취미 모임, 사내연애 등 경로별 만남
- 여러 후보의 이름, 나이, 직업, 성격과 개별 초상화
- 대화 방식과 데이트 선택에 따른 성공·보통·실패
- 호감도·신뢰·집착·도덕성·죄책감 수치와 관계 변화
- 만남 이후 친구, 연인, 가벼운 관계 중 선택
- 호감도, 고백, 연애 여부 선택, 결혼과 이혼
- 양다리·불륜 발각, 평판 하락, 위자료와 직장 해고 위험
- 헤어진 상대 기록과 재회
- 상대 성격·직업에 따른 대화와 관계 사건
- 연인의 경제 성향: 지원형, 각자 관리형, 지출 유발형
- 의료직 당직, 사업자 운영자금, 크리에이터 출연 등 연인 직업별 선택 이벤트
- 친구·연인에게 돈, 직업상 도움, 비밀, 위험한 알리바이 등을 요구하는 행동
- 조건부 행동으로만 처음 만나는 경찰관, 불안정 애착형 일러스트레이터, 재벌 상속인
- 집착 단계별 경고 사건과 감금 배드엔딩
- 집착이 높을 때 장중 메시지, 미행성 방문, 일정·예약주문 방해
- 범죄·거짓말·양다리·선행에 따른 도덕성 변화와 성격별 죄책감·다툼
- 지갑, 책임 전가, 내부정보, 고백, 쉬운 돈 등 도덕성 선택 이벤트
- 특별 히로인 전용 초상화와 일반 직업군 공유 모브 초상화
- 강유진·윤세라·한채린과 일반 모브까지 동일한 2D 캐릭터 시트 화풍 및 4종 감정(기본·기쁨·슬픔·분노) 적용
- 나래의 투자 교육·관계 경계 이벤트와 장태식의 채권 추심·의리 시험 등 특별 인물 전용 선택 사건
- 연인 직업별 기본 소득 재조정과 플레이 연차에 따른 연 4.5% 명목소득 상승(최대 8배)

### AI 라이벌

- 성향이 다른 여러 AI의 현금·보유 종목·평단가·순자산 추적
- AI 사이의 투자 경쟁과 공격
- 플레이어를 향한 합법·불법 경쟁 행동
- 첫 직접 공격 이후 나래의 합법 대응과 장태식의 조언으로 세력을 창설하는 캠페인
- 합법 투자연합·정보/사업 연합·지하 세력 중 조직 노선 선택
- 역공으로 경쟁 세력의 현금·사업가치를 낮추고 순자산 랭킹 1위를 노리는 메인 목표
- 랭킹 1위 달성 시 조직 노선에 따른 세력전 메인 엔딩
- 라이벌 동향 로그와 순자산 비교 차트
- 불법 행동 적발 시 수사와 재판 연계

### 편의 기능

- 나래 NPC의 단계별 튜토리얼과 주식 용어 설명
- 중요한 사건의 선택형 팝업과 상황별 이미지
- 모바일 장 진행 시간 표시
- 관심 종목과 뉴스에서 차트로 바로 이동
- 효과음, 배경음악, 일시정지와 1·2·4배속
- 장중 틱·서킷브레이커·이벤트까지 복원하는 localStorage 자동 저장
- 세이브를 덮어쓰지 않는 읽기 전용 결과 URL 공유
- 업적 시스템

## 실행 방법

별도 빌드 과정이 없는 정적 웹 프로젝트입니다. `index.html`을 직접 열거나 로컬 웹 서버로 실행할 수 있습니다.

```bash
python -m http.server 8000
```

이후 브라우저에서 `http://localhost:8000`으로 접속합니다. 외부 CDN을 사용하므로 인터넷 연결이 없으면 98.css, 차트 또는 축하 효과가 표시되지 않을 수 있습니다.

## 단축키

- `B`: 매수
- `S`: 매도
- `Space`: 일시정지
- `↑` / `↓`: 종목 이동

```text

js/core/trading.js         매수·매도·공매도·지정가 주문 체결 회계

js/companies.js            종목·ETF·경제자산 데이터
js/economy.js              경제 국면, 기준금리, 물가, 자산 영향
js/jobs.js                 직업 데이터

js/rivals.js               AI 라이벌과 경쟁 행동
js/faction_campaign.js      첫 피습·세력 창설·라이벌 제압·세력 엔딩
js/justice.js              수사와 재판
js/housing.js              주거와 부동산
js/life_finance.js         세금·보험·연금
js/health.js               건강과 노화
js/family.js               결혼·자녀·가족
js/social_network.js       인맥과 평판

```

```bash
node tests/core_regression.test.cjs
```
